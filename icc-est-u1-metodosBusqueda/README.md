## Descripción

Este programa permite implementar el método de búsqueda binaria utilizando objetos de tipo (persona) con los atributos nombre y edad. El codigo solicita ingresar un arreglo de personas por teclado, ordena las edades de manera ascendente y posteriormente realiza una búsqueda binaria para encontrar una persona mediante su edad.

Al momento de ejecutar se muestra:

1- La lista de edades arreglada.
2- Los valores de bajo, alto y centro.
3- El valor encontrado en el centro.
4- Si la búsqueda continúa hacia la izquierda o derecha.
5- Si el elemento fue encontrado o no.
